template<class QPXL, typename QSTEP>
class hermes::IntegratorTemplate< QPXL, QSTEP >

Defines the main integrator interface and typical methods shared across all integrators.

Template Parameters
QPXLA type of pixel which an integrator returns (e.g., units::QTemperature, units::QDiffIntensity)
QSTEPA physical quantity (a parameter) that describes a specific map (e.g., units::QFrequency, units::QEnergy)
